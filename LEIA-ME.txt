COMO ATUALIZAR MANUALMENTE
--------------------------
Duplo-clique em "Atualizar Dashboard OQC.bat".
(Le a planilha, refaz o painel e envia para o GitHub/site.)

ATUALIZACAO AUTOMATICA (ja configurada)
---------------------------------------
Tarefa Agendada do Windows: "OQC - Atualizar Dashboard"
Roda DUAS VEZES por dia: 08:15 e 13:15.
A cada execucao ela:
  1) le a planilha (via atalho .lnk);
  2) regenera dashboard.html / index.html;
  3) faz commit e push para o GitHub -> o site online se atualiza
     sozinho em ~1 minuto.
Requer o usuario logado no Windows.

Para mudar horarios / desativar:
  Agendador de Tarefas do Windows -> "OQC - Atualizar Dashboard"
  -> Propriedades (aba Disparadores) ou Desabilitar/Excluir.
